# Groopy

Groopy는 누구나 쉽게 AI 팀을 만들고 활용할 수 있는 노코드 플랫폼입니다. 이 플랫폼을 통해 사용자는 다양한 작업을 자동화하고, 자신의 AI 팀을 공유하거나 판매하여 수익을 창출할 수 있습니다.
